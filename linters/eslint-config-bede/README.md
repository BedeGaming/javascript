# Bede ESLint Config

## Usage

`npm install --save-dev @bedegaming/eslint-config-bede`

Put the following in `.eslintrc`:
```
{
  "extends": "@bedegaming/eslint-config-bede"
}
```

## Text Editors
### Atom
```
apm install linter-eslint
```
